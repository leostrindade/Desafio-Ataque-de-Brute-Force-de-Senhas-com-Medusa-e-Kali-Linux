# Desafio-Ataque-de-Brute-Force-de-Senhas-com-Medusa-e-Kali-Linux
Projeto focado em praticar ataques de força bruta, entender seus impactos e documentar mitigação. Inclui configuração de VMs (Kali + Metasploitable), testes em FTP, DVWA e SMB, além de wordlists, comandos, validações e recomendações de segurança.
